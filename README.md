
Proyecto integrado de DAW curso 2024/2025.

Plataforma marketplace para comprar productos ecológicos 
directamente del agricultor.

El deploy de Vercel es solo el frontend estático de una 
fase anterior que se empezó en el master. El backend con login, base de datos y pagos 
está en desarrollo.

- Frontend: HTML + CSS + JavaScript
- Backend: Node.js + Express  
- Base de datos: Supabase
- Pagos: Stripe
